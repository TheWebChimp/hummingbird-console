a console plugin for hummingbird

Terminals are sexy, it’s not a secret. Well... hummingbird is not a stranger to sexy!
So, we present you console, hummingbird’s plugin to interact with all the awesomeness
of hummingbird, MVC, NORM and CROOD directly from you project instalation.

Console lets you test your favorite models, letting you
execute practically any safe PHP command.

It’s password protected, and it’s super fun-easy to install...
Just download it and put it in your hummingbird plugins folder
or, if you want to continue the terminal nostalgia, install it
with the magnificent and all-powerfull hummingbird cli tool...
